第十一关 基于错误的POST单引号字符型注入

已经显示输入框了，说明是POST提交方式的注入

注入点判断

  1. 在输入框中输入单引号报错，说明为简单的字符型注入
  2. 根据之前GET闯关注入的经验，只是换成post提交
  3. 查看字段数，判断为2个字段

    uname=admin’ order by 2#&passwd=&submit=Submit 正常
    uname=admin’ order by 3#&passwd=&submit=Submit 不正常

  4. 直接上payload

payload直接查出数据库所有数据
